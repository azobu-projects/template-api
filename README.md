## Getting Startedf

### Installation

Install dependencies using `yarn`:

```sh
yarn
```

### Run

```sh
yarn dev
```

Access it on <http://localhost:3000>.

### Lint

```sh
yarn lint
```

To fix basic issues:

```sh
yarn lint --fix
```

### Test

```sh
yarn test
```
